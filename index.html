Discord with the power of C++

Thanks to C++, Sleepy Discord is able to be fast and efficient while also allowing you to program with multiple paradigms. C++ is often the language of choice for developing for multiple devices. With this fact in mind, Sleepy Discord was designed to work with more devices then any other before it.

Live Demo

Portable

As long as your platform or environment can use the internet, Sleepy Discord can run on it.

No other libraries are required

Besides the C++ Standard Library, Sleepy Discord can used without any other libraies. By not doing any input or output operation by itself, Sleepy Discord can work across many platforms and I/O frameworks. Sleepy Discord also has build in support for CPR and WebSocket++ to get you quickly started.
