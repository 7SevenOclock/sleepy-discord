SleepyDiscord::DiscordClient::pinMessage

DiscordClient > pinMessage

Version as of Writing: Prerelease Date: 3/9/2017

Declared in client.h and defined in client.cpp 
bool pinMessage(std::string channel_id, std::string message_id);
Pins a message in a channel. You'll need permission MANAGE_MESSAGES to do this. Uses Add Pinned Channel Message

Parameters

channel_id The id of the channel you want to pin the message to
message_id The id of the message you want to pin

Return value

true on success, otherwise false

Example

#include <sleepy_discord>

int main() {
    myClientClass client("token");
    client.waitTilReady();
    client.pinMessage("channel id", "messagee_id");
}

Output: message pinned
